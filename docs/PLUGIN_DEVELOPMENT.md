# Plugin Development

Sky Weaver uses a safer Allsky-inspired module system. Built-in modules are trusted. Uploaded custom modules are disabled until an admin explicitly enables them and should be treated as local code execution.

Planned triggers:

- `daytime_capture`
- `nighttime_capture`
- `day_to_night_transition`
- `night_to_day_transition`
- `periodic`
- `manual`
